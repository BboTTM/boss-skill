# 老板.skill PRD 摘要

## 目标

做一个“老板模拟器” skill，让用户可以：

1. 快速创建一个老板角色
2. 直接进入高压对话演练
3. 随着更多资料和纠正不断进化角色
4. 在角色跑偏时回滚到旧版本

## 用户路径

### 路径 A：快速演练

1. `/create-boss`
2. 选择 `archetype`
3. 生成 `/{slug}`
4. 开始演练

### 路径 B：真人复刻

1. `/create-boss`
2. 选择 `real-person`
3. 导入聊天、文档、截图、描述
4. 生成 `/{slug}`
5. 演练中持续纠正

## 核心能力

- 老板 archetype 生成
- 真人资料抽取
- 资料归档
- 纠正进化
- 版本回滚
- 沉浸式 / 复盘式切换

## 非目标

- 现在还不做完整 OCR 引擎
- 现在还不做多平台自动采集器
- 现在还不做 UI 面板

## 当前实现状态

- 已有可运行的 meta-skill
- 已有资料分析脚本
- 已有角色写入、导入、纠正、回滚脚本
- 已有单元测试覆盖主链路
