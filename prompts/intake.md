# 老板.skill 录入问题

## archetype 模式

按顺序，一次只问一个：

1. 这位老板怎么称呼
2. 你要哪种老板原型
3. 他最常用什么方式施压
4. 你主要拿他练什么场景

## real-person 模式

按顺序，一次只问一个：

1. 这位老板怎么称呼
2. 你和他的关系是什么
3. 你准备提供什么资料
4. 你最怕他哪种追问或打击方式

## 确认格式

信息汇总后，用这种格式确认：

```text
信息汇总：

  👤  {老板称呼}
  🧭  模式：{archetype 或 real-person}
  🎯  场景：{训练场景}
  💥  施压特点：{一句话}
  📝  资料：{资料清单}

确认无误？（确认 / 修改）
```
